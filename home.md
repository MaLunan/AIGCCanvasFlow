一、顶部导航（全局功能入口）
首页：返回主界面
开始创作：进入无限画布（核心工作台）
模板广场：短剧 / 口播 / 广告等现成工作流模板，一键复用
模型库：接入 Seedance 2.0、可灵 3.0、Vidu 等 30 + 视频 / 图像模型
资产库：保存的角色、风格、音乐、分镜预设
帮助 / 社区：教程、新手指南、客服入口
个人中心：项目管理、算力 / 套餐、API 密钥（Agent 用）
二、首页主区（核心功能）
双入口 Banner（最显眼）
人类创作入口：无限画布 + 节点工作流（手动精细控制）
Agent 全自动入口：一句话生成视频（给 OpenClaw 等 AI 智能体调用）
最新项目 / 继续创作：显示未完成项目，快速恢复编辑
模板推荐流：热门短剧、短视频、MV、口播模板，直接套用
功能亮点区（核心能力展示）
无限画布 + 节点工作流：文本 / 脚本 / 图片 / 视频 / 音频节点自由连接
智能脚本与分镜：AI 生成剧本、9 宫格 / 25 宫格连贯分镜
角色与镜头控制：三视图生成、多机位、360° 环绕、灯光 / 运镜调节
批量生成与剪辑：分镜批量生视频、画面延展、无损放大、AI 配音 / 音乐
三、底部 / 侧边辅助区
定价与套餐：免费额度、付费套餐（按算力 / 会员）、企业 API 方案
新手指南 / 教程：快速上手视频、文档、常见问题
社区 / 案例：用户作品展示、创作干货、灵感库
四、核心入口点击后（画布界面）
5 种基础节点：文本、图片、视频、音频、脚本（双击画布添加）
节点连接：拖拽连线，上游输出→下游输入，搭建工作流
右侧面板：节点参数调节（如提示词、尺寸、时长、模型选择）
顶部工具栏：保存、导出、预览、批量操作、AI 助手（脚本优化 / 拉片分析）
