# AIGCCanvasFlow — 已实现功能清单

> 技术栈：Vue 3.5 + Vite 5 + Pinia 3 + @vue-flow/core 1.48
> 主要工作目录：`/h5`

---

## 一、画布基础能力

| 功能 | 说明 |
|------|------|
| 无限画布 | 支持任意方向平移、缩放（0.1x ~ 4x） |
| 双层网格背景 | 线条网格 + 点阵网格叠加，可独立开关 |
| 吸附网格 | 可配置网格大小，工具栏 ON/OFF 切换 |
| 地图式手势 | 单指/鼠标拖拽平移；双指捏合缩放；触控板双指滑动平移 |
| 框选节点 | `Shift` + 拖拽多选 |
| 小地图 | 右下角 MiniMap，实时反映画布内容 |
| 适应视图 | 快捷键 `F` 或工具栏按钮，动画过渡居中显示全部节点 |

## 二、节点系统

### 节点类型

| 类型 | 说明 |
|------|------|
| **TextNode** 文本节点 | 双击进入编辑，Ctrl+Enter/Escape 退出；内容输出到连线标签 |
| **ImageNode** 图片节点 | 上传图片（本地文件），hover 显示遮罩，可换图/清除；图片按原始宽高比显示 |
| **VideoNode** 视频节点 | video.js 播放器，支持上传本地 MP4/WebM/OGG/MOV；可换文件/清除 |
| **NoteNode** 备注节点 | 便利贴样式，双击编辑，5 色切换，可拖拽调整大小 |
| **GroupNode** 分组节点 | 可拖拽调整大小的容器，子节点跟随父节点移动 |

### 节点通用功能

- **NodeHeader**：双击标签名可重命名；点击类型徽章可切换节点类型（文本/图片/视频）
- **NodeAddButton**：每个节点右侧的 `+` 按钮，点击弹出类型选择器，自动在右侧空位创建并连线
- **删除**：节点头部 `×` 按钮、右键菜单、`Delete`/`Backspace` 快捷键
- **复制**：右键菜单 → 复制节点

## 三、连线系统

### 连线规则

- 仅允许 **头（source）→ 尾（target）** 单向连接
- **头→头** 自动纠正：保持方向 A→B，目标端自动改为 tl（尾）
- **尾→尾** 自动纠正：方向反转为 B→A，两端 handle 自动修正

### 连线创建方式

| 方式 | 说明 |
|------|------|
| 拖拽 handle | 从节点右侧 source handle 拖出，连接到目标节点左侧 target handle |
| 拖拽到空白区 | 拖出后松手弹出节点类型选择器，创建新节点并自动连线（Edge-Drop Picker） |
| NodeAddButton | 点击 `+` 按钮选择类型，在右侧空位创建节点并连线 |

### 连线样式

- 贝塞尔曲线 + 箭头终点 + 动画流动效果
- 选中高亮，右键可切换动画/删除

### 数据流

- 节点 `outputValue` 更新时，自动同步到所有出边的标签（`edge.label`）

## 四、图片节点 — AI 生成面板

节点内置「上传 / AI 生成」双模式切换标签：

**上传模式**
- 点击占位区或换图按钮上传本地图片
- hover 遮罩：「换图片」「✕ 清除」

**AI 生成模式**
- 提示词文本框
- 比例选择：`1:1` / `4:3` / `3:4` / `16:9` / `9:16`
- 模型选择：Flux / DALL·E 3 / SD 3.5 / Midjourney / Ideogram
- 生成按钮（含 loading 旋转动画，空 prompt 时禁用）
- ⚠️ 生成逻辑为预留接口，需接入实际大模型 API

## 五、视频节点 — 功能

### 播放器

- video.js 播放器，支持本地 MP4/WebM/OGG/MOV
- 换文件 / ✕ 清除

### 帧提取与截图

- 底部帧条（VideoFrameStrip）：自动逐秒截帧，横向滚动缩略图
- 点击帧缩略图：跳转播放器到对应时间点
- 点击帧上的生成按钮：高清截取该帧，**自动在画布右侧创建 ImageNode 并连线**（遵循同款位置碰撞算法）

### AI 生成面板

节点内置「上传 / AI 生成」双模式切换标签：

**AI 生成模式**
- 提示词文本框
- 比例选择：`16:9` / `9:16` / `1:1` / `4:3` / `3:4`
- 模型选择：Kling / Sora / Runway / Pika / Hailuo
- 时长选择：`3s` / `5s` / `10s` / `15s` / `30s`
- 清晰度选择：`480p` / `720p` / `1080p` / `4K`
- 音频切换：🔊 有声 / 🔇 无声（按钮组）
- 生成按钮（含 loading 动画）
- ⚠️ 生成逻辑为预留接口，需接入实际大模型 API

## 九、节点分组

- 选中 2 个及以上节点，点击工具栏「分组」按钮
- 创建 GroupNode 容器包裹所有选中节点
- 子节点跟随分组移动，限制在分组范围内拖拽
- 分组可拖拽调整大小（NodeResizer）

## 十、已修复的关键技术问题

| 问题 | 修复方案 |
|------|----------|
| 新建节点后已拖拽节点位置弹回 | 写入 store 前先从 `getNodes.value` 快照当前坐标，原子更新 |
| 切换节点类型后位置弹回 | NodeHeader 调用前通过 `findNode` 读取实时坐标传入 `changeNodeType` |
| 截帧生成图片放在节点下方 | 改用 `findFreePosition` 算法放置到右侧，与 NodeAddButton 逻辑一致 |
| 头→头/尾→尾 非法连线 | `onConnect` 中检测 handle 类型，自动纠正为合法的 source→target |
